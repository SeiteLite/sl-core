# sl-core

The core part of the SeiteLite framework.

## Directory structure

```bash
├── assets/   # logo
├── cls/      # php autoloaded classes
├── els/      # custom elements
├── inc/      # include files
├── pages/    # error pages
├── tpl/      # page templates
├── ts/       # TypeScript source
└── ...       # etc.
```
